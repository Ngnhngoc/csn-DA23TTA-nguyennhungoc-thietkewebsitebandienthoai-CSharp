# Đồ án Thiết kế Website Bán Điện Thoại

## Thông tin nhóm tác giả
- **Họ và tên**: Nguyễn Như Ngọc (Ví dụ)
- **Email**: [Email của bạn]
- **Điện thoại**: [Số điện thoại của bạn]

## Cấu trúc thư mục
- **setup**: Chứa các script SQL để khởi tạo cơ sở dữ liệu.
- **src**: Chứa mã nguồn chương trình (Solution và Project).
- **progress-report**: Báo cáo tiến độ.
- **thesis**: Tài liệu đồ án (Báo cáo, Slide, v.v.).
- **soft**: Phần mềm liên quan.
- **docker**: Cấu hình Docker.

## Hướng dẫn cài đặt
1. Mở `src/webbandt.slnx` bằng Visual Studio.
2. Cấu hình chuỗi kết nối trong `appsettings.json`.
3. Chạy các script trong thư mục `setup` để tạo database.
4. Build và chạy dự án.
